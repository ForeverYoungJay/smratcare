# 家属端小程序真机验收清单

## 发布前准备
- 微信公众平台已添加 `https://gfyy.org.cn` 到小程序 `request 合法域名`
- 小程序默认接口地址为 `https://gfyy.org.cn`
- 服务器 `https://gfyy.org.cn/api/auth/family/bootstrap` 可返回 `code=0`
- 当前未接真实短信时，验证码统一使用 `1234`

## 登录与账号
- 首次注册可成功提交并进入首页
- 找回密码可用验证码 `1234` 重置成功
- 密码登录成功后再次进入小程序保持登录态
- 登录失效后会跳回登录页，不出现白屏

## 关键业务链路
- 首页可正常加载老人信息、消息摘要、快捷入口
- 在线缴费页可正常打开，不出现接口域名错误
- 评估报告页可正常进入，失败时有提示
- 在线沟通页可正常打开，上传失败时有提示
- 帮助中心、个人中心、安全设置可正常加载

## 环境与安全
- 真机环境不再出现 `fail url not in domain list`
- 小程序运行提示显示为 HTTPS 正式域名
- 服务器不再使用默认 JWT 密钥
- 宿主机 Nginx 反代 `/` 到 `127.0.0.1:8081`
- 宿主机 Nginx 反代 `/api/`、`/ws/`、`/uploads/` 到 `127.0.0.1:8080`

## 上线前复核
- iOS 微信真机通过
- Android 微信真机通过
- 弱网下首页、登录、注册、找回密码都有明确错误提示
- 微信后台已完成提审所需的隐私与域名配置
