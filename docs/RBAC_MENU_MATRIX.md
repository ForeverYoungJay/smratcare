# RBAC 菜单矩阵导出说明

## 目的

快速产出“不同角色能看到哪些一级菜单”的矩阵，方便：

- 权限评审会
- 部门培训
- 上线前权限核对

## 执行命令

```bash
./scripts/rbac_menu_matrix.sh
```

## 输出形式

脚本会输出一段 Markdown 表格（角色 × 一级菜单），可直接复制到：

- 企业微信群公告
- 项目周报
- 验收文档

## 适用角色

- 六部门员工/部长
- `DIRECTOR`
- `SYS_ADMIN`

## 注意

- 该脚本基于前端路由 `meta.roles` 与权限映射规则静态计算，不依赖后端在线状态。
- 若你调整了路由角色配置，请重新执行导出，避免使用旧矩阵。
